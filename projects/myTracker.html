My Tracker

January 21, 2021 | Cross Platform Application | Open My Tracker

My Tracker is an time tracking application with the cloud storage capability. Users can sign up and login with any web browsing devices and track their own productivity related to any category that they added.

Feature lists:

  • Ability to create task's category
  • Start Tracking / Stop Tracking
  • Enter short description of task
  • Select which task's category
  • Store remotely
  • View the list of tracks
  • Open and check the detail of the tracks

Techs I used to implement:

  • React: Javascript Framework
  • Ionic: UI Components
  • Capacitor: Native functionality of devices
  • Firebase: Authentication / Storage
  • Vercel: Hosting
  • Undraw: SVG Illustration
